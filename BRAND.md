You may use the AgentTrader name when it is necessary to describe the origin of this repository or compatibility with the project, for example:

- "forked from AgentTrader Public Arena"

Such references must be accurate and must not imply sponsorship, endorsement, partnership, or official status.

- remove or alter attribution notices in a way that misrepresents the origin of the project
